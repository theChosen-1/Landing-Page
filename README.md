# Landing Page

A simple, responsive landing page built as part of [The Odin Project](https://www.theodinproject.com/) curriculum.

## Live Demo

[View Live Site](https://thechosen-1.github.io/Landing-Page/)

## About

This is the first project from The Odin Project's Foundations course. It's a static landing page that demonstrates fundamental HTML and CSS skills including:

- Semantic HTML structure
- CSS Flexbox for layout
- Responsive design principles
- Basic styling and typography

## Built With

- HTML5
- CSS3

## The Odin Project

The Odin Project provides a free, open-source curriculum for learning full-stack web development.

## What I Learned

- Structuring a webpage with semantic HTML
- Using CSS Flexbox for page layouts
- Styling elements with colors, fonts, and spacing
- Linking external stylesheets
